**Question**: Prove ¬P → Q ≡ P ∨ Q using a truth table.
**Solution**: | P     | Q     | ¬P    | ¬P → Q | P ∨ Q |
|-------|-------|-------|--------|-------|
| True  | True  | False | True   | True  |
| True  | False | False | True   | True  |
| False | True  | True  | True   | True  |
| False | False | True  | False  | False |
The columns for ¬P → Q and P ∨ Q are identical, so they are equivalent.

**ExerciseType**:   
**Question**: Is ∀x ∈ ℕ, ∃y ∈ ℕ, x < y true? Explain.
