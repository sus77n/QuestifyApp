## EX S.15
**ResponseType**: Short Answer  
**Difficulty**: 3  
**ExerciseType**:   
**Question**: Prove that (P → Q) ∧ (Q → R) is logically equivalent to P → (Q ∧ R) using logical laws.
**Solution**: 
- Step 1: Rewrite P → Q as ¬P ∨ Q and Q → R as ¬Q ∨ R.
- Step 2: (¬P ∨ Q) ∧ (¬Q ∨ R).
- Step 3: Apply Distributive Law: (¬P ∧ ¬Q) ∨ (¬P ∧ R) ∨ (Q ∧ ¬Q) ∨ (Q ∧ R).
- Step 4: Since Q ∧ ¬Q is a contradiction (false), it simplifies to (¬P ∧ ¬Q) ∨ (¬P ∧ R) ∨ (Q ∧ R).
- Step 5: Factorize: ¬P ∧ (¬Q ∨ R) ∨ (Q ∧ R).
- Step 6: Since ¬Q ∨ R ≡ Q → R, the expression becomes ¬P ∨ (Q ∧ R), which is P → (Q ∧ R).

**Question**: Translate the predicate logic statement ∀x ∃y (x + y = 0) into a natural language sentence, where x, y ∈ ℝ.

## EX S.17

**Difficulty**: 3  
**ExerciseType**:   
**Question**: Express the statement "Every student has a unique advisor" using quantifiers, where S(x) means "x is a student," A(y) means "y is an advisor," and H(x, y) means "y is the advisor of x."
**Solution**: ∀x (S(x) → ∃!y (A(y) ∧ H(x, y))), where ∃! denotes "there exists a unique."

**Difficulty**: 3  
**ExerciseType**:   
**Question**: Determine whether the statement ∀x ∃y (x < y) is true for x, y ∈ ℕ, and explain why.
**Solution**: The statement is true. For every natural number x, there exists a natural number y such that x < y (e.g., y = x + 1). Since the natural numbers are unbounded above, such a y always exists.
